Instructions

The instructions for downloading SODA10M is listed as follows:

  1. All compressed files need to be placed in the same folder for decompression, the standard path has been compressed into the file.
  2. For each unlabeled image split, please "cat" the different parts into one tar first.
  3. The SODA10M test set data will be released during the ICCV2021 workshop.

After decompression, the data organization is listed as follows:

"SODA10M": {
        |---SSLAD-2D
            |---Labeled
                |---train
                |---val
                |---test
                |---annotations
                    |---instance_train.json
                    |---instance_val.json
                    |---instance_test.json
            |---Unlabel
                |---images
                    |---images_*
                |---annotations
                    |---instance_unlabel_*.json
        }
